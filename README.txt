En el cuaderno de Python grafos.ipynb se puede observar detalladamente el proceso de implementación de la nueva funcionalidad estudiada en la memoria del proyecto. En el cuaderno todas las celdas se encuentran ejecutadas, con ejemplos para todas las funciones creadas. Si se desea comprobar el funcionamiento de dichas funciones, serán necesarias dos cosas:

- Primero habrá que descargar el archivo kg.csv del siguiente enlace https://dataverse.harvard.edu/file.xhtml?fileId=6180620&version=2.1 donde se encuentran todos los nodos y relaciones de la ontología PrimeKG, ya que el archivo es demasiado grande para subir al repositorio de GitHub.

- Lo siguiente será introducir una llave para la API de OpenAI en el archivo .env, ya que algunas de las funciones hacen llamadas a una instancia de ChatGPT.

Una vez descargados todos los archivos en la misma carpeta, el cuaderno debería ejecutarse sin problemas.
